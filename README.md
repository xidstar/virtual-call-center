# Design Technologist ‚Äì LLM Workflow Builder

This project is a small React application that implements a visual workflow builder for defining the sequence of questions an LLM agent will ask during insurer phone calls.

The goal of the exercise was to translate a provided Figma design into a functional MVP UI with clear structure, predictable behavior, and clean, readable code.

---

## ‚ú® Features

- Linear workflow of **question nodes**
- Visual **connectors** between nodes to indicate sequence
- **Add Node** button appends a new question to the end of the workflow
- Static, predefined question content (per MVP requirements)
- Styling aligned closely to the provided Figma design
- Built with modern React and Tailwind CSS
- Lucide icons used for all iconography
